# 00. 世界观与核心循环

## 0.1 世界观前提

Factor 不是单纯能源，而是“现实稳定度”的表现。古代文明 **Arbiter** 曾建造跨维度巨构维持平衡，最终因过度提取导致崩解。玩家作为“继承者”，在“收益最大化”与“生态稳定化”之间持续抉择。

## 0.2 四阶段核心循环

1. **适应（Survival）**：观测潮汐，选择高低能窗口行动。  
2. **交互（Interaction）**：建立基础机器，学会能量缓存与释放。  
3. **掌控（Control）**：搭建多方块结构，实现区块级稳定。  
4. **重构（Reshape）**：跨维度产业链与服务器级生态治理。

## 0.3 主线章节（建议）

1. 听见潮汐（教学：观测）
2. 第一次失稳（教学：灾害应对）
3. 维度开口（教学：跨维度传输）
4. 古代回声（教学：遗迹与蓝图）
5. 最终抉择（稳定结局 / 重塑结局）

## 0.4 阵营（可选）

* **守衡者**：偏修复、防护、农业增益。  
* **裂潮者**：偏爆发、过载战斗、极限产能。  
* **虚序会**：偏存储、冻结、精密控制。

阵营提供独占任务线、装备模组分支和结局文本差异。
