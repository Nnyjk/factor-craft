# 04. 战斗、武器、防具、饰品与特效强化

## 4.1 武器体系（新增特化）

### 专属武器

* **潮汐刃（Tideblade）**：根据当前 Tier 切换攻击模式（稳定=均衡、过载=爆发）。
* **裂隙长枪（Rift Lance）**：穿透与位移技，适合突进控制。
* **仲裁法器（Arbiter Focus）**：施加区域稳压/失稳场，偏团队辅助。

### 武器特效强化

* **飞行突击（Aerial Dash）**：短 CD 空中冲刺（需饰品供能）。
* **裂隙跃迁（Rift Blink）**：短距离瞬移并留下减速场。
* **潮汐超载（Tide Overdrive）**：短时攻速与伤害大幅提升，结束后虚弱。

## 4.2 防具细节扩展

### 套装定位

* **守衡者套**：高防御、团队减伤、净化导向。
* **裂潮者套**：高爆发、反伤、过载收益放大。
* **虚序会套**：护盾与位移，偏机动控制。

### 防具部位机制

* **头盔**：侦测/显示下一次日切判定趋势。
* **胸甲**：核心技能位（护盾、短滞空、稳压场）。
* **护腿**：地形适应（熔岩边缘、低重力）。
* **靴子**：冲刺、缓降、空中二段跳。

### 防具词缀与强化

* **日蚀护层**：夜间抗性提升。
* **晨光重启**：日切后首次技能无消耗。
* **谐波回充**：稳定期自动回复少量耐久与能量。

## 4.3 饰品系统（扩展）

### 饰品槽位

* 核心饰品（1）
* 功能饰品（2）
* 维度印记（1）

### 核心饰品示例

* **浮空棱镜**：提供短时飞行（悬停 3~6 秒）。
* **裂隙罗盘**：标记附近裂隙事件与稀有资源点。
* **稳压徽章**：降低过载 Debuff 强度。

### 功能饰品示例

* **回响护符**：击杀后恢复少量护盾。
* **导因指环**：提高机器交互速度与法术导因率。
* **虚序怀表**：延长 BUFF 持续，缩短单次位移技能冷却。

### 维度印记

* **下界印记**：火系强化与灼烧抗性。
* **末地印记**：位移与控制强化。
* **主世界印记**：回复与资源效率强化。

## 4.4 特殊能力：飞行与机动

为避免前期失衡，飞行能力分层开放：

1. **滑翔阶段**：靴子 + 饰品实现缓降与短滑翔。
2. **悬停阶段**：胸甲技能提供短时悬停（冷却较长）。
3. **战术飞行阶段**：高阶饰品+套装联动，开启短时自由飞行。
4. **终局飞行阶段**：仲裁级装备在特定 Tier 窗口可持续飞行。

> 飞行状态受“导因率消耗”与“当前 Tier”双重约束，避免常驻飞行破坏探索节奏。
