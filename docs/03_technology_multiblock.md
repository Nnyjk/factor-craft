# 03. 科技树、多方块与模块强化

## 3.1 核心设备

* Factorometer（因子计）
* Resonator（共振器）
* Stabilizer（稳定锚）
* Rift Projector（裂隙投射器）

## 3.2 多方块机器

1. **谐波铸造阵**：批量冶炼、连铸高吞吐。  
2. **以太蒸馏塔**：液态 Factor 多级蒸馏。  
3. **裂隙织机**：维度签名与部件绑定。  
4. **虚序封存库**：高危能量封存与削峰填谷。  
5. **潮汐反应釜**：终极窗口合成。

## 3.3 多方块强化模块

### 结构模块
* Reinforced Frame：结构容错 +
* Adaptive Brace：过载崩溃概率 -

### 流程模块
* Parallel Chamber：并行通道 +1~+3
* Catalyst Matrix：工艺时长 -，催化剂消耗 +

### 稳定模块
* Phase Damper：输入波动 -，峰值效率 -
* Entropy Sink：异常脉冲吸收

### 维度模块
* Nether Manifold：热工/战利品收益 +
* End Lattice：精密合成成功率 +

> 模块叠加受“功率预算上限”限制，超限将降频运行。

## 3.4 多方块升级路线

* T1：单核心 + 低级框架
* T2：双流程舱 + 基础稳定舱
* T3：维度模块槽解锁
* T4：跨维度联动 + 日切自动调优
* T5：仲裁协议接入（终局）
